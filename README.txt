## Getting Started

To get started with this project, please follow the instructions below:

### Prerequisites

Before you begin, make sure you have the following installed on your computer:

- Node.js
- npm

### Installing

1. Unzip the `project.zip` file.

2. Navigate to the project directory in your terminal by running the command:
    
    ```bash
    cd abhinav-portfolio
    ```

3. Install the dependencies by running the command:
    
    ```bash
    npm install
    ```

### Building and Running the Project

1. Build the project by running the command:

    ```bash
    npm run build
    ```

2. Run the project by running the command:

    ```bash
    npm start
    ```


3. Access the project by opening a web browser and navigating to:

    ```bash
    http://localhost:3000
    ```
